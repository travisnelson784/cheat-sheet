HTML elements cheat sheet

HTML-HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content.

HTML element-An HTML element is a piece of content in an HTML document and uses the following syntax: opening tag + content + closing tag. In the code provided:

HTML attributes-HTML attributes are values added to the opening tag of an element to configure the element or change the element’s default behavior.

  • class=""- attribute specifies one or more classnames for an element. can also be use to in evry html element. attribute is mostly used to point to a class in a style sheet.
  • id=""-attribute specifies a unique id for an HTML element. can be used in every element. attribute is most used to point to a style in a style sheet.
  • onclick-attribute is part of the Event Attributes, and can be used on any HTML elements. can be used on all element.
